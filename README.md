# Compare Parser Performance String vs Regex

This example demonstrates how different parsing strategies can significantly impact data pipeline performance. It compares a regex-based parser (often slower for simple delimited data) with a string-splitting parser, processing a large simulated dataset. The output shows the time taken by each parser, highlighting the performance bottleneck often caused by inefficient parsing.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
